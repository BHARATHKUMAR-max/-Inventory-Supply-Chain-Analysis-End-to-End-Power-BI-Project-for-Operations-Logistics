# 📦 Power BI Project: Inventory & Supply Chain Analysis | End-to-End Dashboard

Managing inventory and ensuring an efficient supply chain are critical for business success. In this Power BI project, we analyze stock management, demand forecasting, supplier performance, and logistics optimization by building a fully interactive dashboard that reflects real-world business operations.

---

## 🚀 Project Highlights

This project demonstrates how Power BI can be used to monitor and improve supply chain performance.

### 🧹 Data Cleaning & Transformation
- Used **Power Query** to clean and transform raw datasets.
- Handled missing values and standardized data formats.
- Created relationships between tables for accurate data modeling.

### 📦 Inventory Metrics
- Track **stock levels**
- Calculate **inventory turnover rates**
- Identify **reorder points**
- Monitor **backorders and stockouts**

### 🚚 Supply Chain KPIs
- Analyze **supplier lead times**
- Measure **delivery performance**
- Evaluate **supplier reliability**
- Track **order fulfillment rates**

### 🔢 Advanced DAX Calculations
- Inventory Turnover
- Average Lead Time
- Reorder Level
- Supplier Performance Score
- Demand Trend Analysis

### 📊 Professional Dashboard Design
- Interactive slicers and filters
- KPI cards for quick insights
- Dynamic charts and visuals
- Executive-level layout for decision makers

### 🗺️ Real-World Data Storytelling
- Convert raw supply chain data into actionable insights
- Present key metrics clearly for business stakeholders
- Highlight operational bottlenecks and improvement areas
